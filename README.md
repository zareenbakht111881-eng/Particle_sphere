# Particle Sphere

An interactive 3D particle sphere controlled by real-time hand tracking using MediaPipe Hands.

## Features

- 3D particle sphere (3000 particles)
- Real-time webcam hand tracking
- Single-hand movement and rotation
- Pinch gesture to charge and trigger particle explosion
- Automatic particle reformation
- Two-hand scaling (grow/shrink)
- Finger count changes particle colors
- Fullscreen webcam background toggle
- Minimal dark UI
- Built with HTML, CSS and JavaScript

## Technologies

- HTML5
- CSS3
- JavaScript
- MediaPipe Hands
- Canvas API
